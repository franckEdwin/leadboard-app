# Leadboard App

The Leadboard App is a simple Flask application for managing a team leaderboard with their points. Administrators can update points and create new teams.

## Dependencies

- Python 3.8+
- Flask 2.0.1+
- Flask-Login 0.5.0+
- Flask-SQLAlchemy 3.0+
- Flask-Migrate 3.0+
